# GombalAI 💘

AI Texting Assistant — bantu cowok bales chat gebetan dengan 3 opsi siap copas.

## Deploy ke Vercel (5 menit)

### 1. Upload ke GitHub
- Buat repo baru di github.com (nama: `gombalai`)
- Upload semua file ini

### 2. Deploy ke Vercel
- Buka vercel.com → New Project
- Import repo `gombalai` dari GitHub
- **PENTING**: Tambahkan Environment Variable:
  - Key: `ANTHROPIC_API_KEY`
  - Value: `sk-ant-...` (API key Anthropic kamu)
- Klik Deploy

### 3. Done!
Dapat link seperti `https://gombalai.vercel.app` — share ke buyer!

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Anthropic Claude API
